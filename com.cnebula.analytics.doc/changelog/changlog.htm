CALIS日志统计与监控系统更新历史

从 build167升级到 build?

  1. 中心站、共享域站、成员馆站点增加今日统计、实时分析功能

从 build130升级到 build167

  1. 图表控件替换为highcharts
  2. 图表支持打印、导出PNG、JPEG、PDF、SVG矢量图功能
  3. 增加实时分析、今日统计功能
  4. 增加受访页面、全部来源、IP来源、外部链接、系统环境报表
  5. 表格支持excel的导出、打印、拷贝功能
  6. 更新日期控件为日期范围选择控件

从 build70升级到 build130

  1. CALIS日志统计与监控系统升级到v1.1.0版本
  2. 支持多指标报表
  3. 支持可查看全站级别的各个应用系统的报表
  4. 增加对教参系统的报表集成
  5. 增加功能菜单的配置管理（conf/menu/menu_config.xml）
  6. 增加对rcs数据的过滤配置管理（conf/rcsFilter/rcsFilter_config.xml）

从 build64升级到 build70

  1. 修复bug1453:日志系统js端114版本某些时候采集_caam的值为0，
  2. 建议日志系统同步数据的地址做成可配置项
  3. 修复bug1841日志系统运行1~2周后不定期出现NPE

升级办法：

在原var.propertes中添加：


#从RCS获取信息的配置
